# zenBBS
A PenguinMod based chat application which aims to provide easy communication for everyone.

zenBBS is currently in alpha, and it really shouldn't be expected to work too well. While there is a relatively secure hash function for passwords, I can't in good faith recommend you to use a password you use elsewhere (which really holds true for everything).

Changelogs and information are currently located in the pmp file (source code) which can be opened using PenguinMod in any common browser.

thank you for using zenBBS :)
