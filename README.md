# zenBBS
A PenguinMod based chat application which aims to provide easy communication for everyone.

zenBBS is currently very early in development., and it really shouldn't be expected to work too well. Thus, it isn't publicly released yet. 

As far as features, it's about a fifth done, but time-wise, I could see a release maybe as soon as June-August 2026 at soonest.

No you cannot test it... Sorry, I'd be more open but I don't want to use GarboMuffin's servers for a public chat. That feels like an overstep.

thank you for your interest in zenBBS :)
